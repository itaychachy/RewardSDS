# RewardSDS: Aligning Score Distillation via Reward-Weighted Sampling

</p>

___

> **TL;DR:** 
> SDS struggles with fine-grained alignment, so we introduce RewardSDS, which weights noise samples based on a reward model to prioritize high-reward outputs. Our method generalizes to VSD, leading to RewardVSD, which improves alignment in text-to-image and text-to-3D tasks. We evaluated RewardSDS and RewardVSD on multiple benchmarks and found they significantly outperform SDS and VSD in both generation quality and alignment.
